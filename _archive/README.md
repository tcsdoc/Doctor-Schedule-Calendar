# PSC Archive

Historical builds, docs, and development artifacts kept for reference. Not used by the active Xcode app target.

- **builds/** — Old `.xcarchive` exports and IPA packaging output
- **development/** — Print UI experiments, data migration tool, utility scripts
- **docs/** — Historical design notes, release write-ups, and troubleshooting history
- **snapshots/** — Marketing and UI reference images (including legacy iPhone shots)

Active project docs remain at the repo root: `PSC_CLOUDKIT_SUPPORT_HANDOFF.md`, `PSC_DUPLICATE_INVESTIGATION.md`, and `PrivacyPolicy.md`.
